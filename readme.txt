== Description ==

Add SlideBox to your Content. 
A button in visual editor avoid to create a slidebox with fancy effect slideup/slidedown.
An configuration page avoid to change images and css of slidebox.

== Installation ==

Installation of the plugin.

1. Upload plugin directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
